## 使用方法

1. 游戏内画质设置为自定义
2. 关闭正在运行的游戏
3. 运行本软件
4. 填入需要修改的参数
5. 点击完成（请允许本软件修改注册表
6. 退出软件
7. 启动游戏

`修改之后不必再运行本软件，若想修改为官方设定，游戏内修改即可`
