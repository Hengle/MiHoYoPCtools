## 使用方法

1. 关闭正在运行的游戏
2. 运行本软件
3. 填入需要修改的参数
4. 点击完成（请允许本软件修改注册表
5. 退出软件
6. 启动游戏

`修改之后不必再运行本软件，若想修改为官方设定，游戏内修改即可`
